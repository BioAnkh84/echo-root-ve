# Roadmap

## v0.1.0 License-Ready MVP

- Installable local demo
- Sample action interception
- `PROCEED`, `PAUSE`, `ABORT`, and `SAFE_MODE` decisions
- Receipt schema
- Hash-chain receipt engine
- Replay verification
- Focused unit tests
- Release evidence report
- Commercial license placeholder

## Next Candidates

- CLI wrapper for structured JSON action input
- Stronger policy file loader
- Optional JSON Schema dependency for strict schema validation
- Richer provider fallback receipts
- Signed release artifacts
- External review packet

## Out Of Scope For v0.1.0

- Certification claims
- Guaranteed compliance claims
- Autonomous destructive operations
- Production deployment support without a separate hardening pass
