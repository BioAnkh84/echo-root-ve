# Security Policy

If you believe you’ve found a security issue in Vulpine Echo (VE) or its kernel:
- **Do not** open a public issue with exploit details.
- Email: security@placeholder.invalid (or use GitHub Security Advisories)
- Include: VE commit/tag, OS + PS version, minimal repro steps.

We will acknowledge within 72 hours and coordinate a fix + disclosure window.

---

## Do Not Submit Private Data

Please do not attach or paste:

- credentials, keys, tokens, or secrets
- private ledgers or operator receipts
- private Echo Nexus / Cipher habitat data
- personal memory exports or chat logs
- medical, legal, financial, or family records
- unreleased internal scoring, policy, or operator workflow details

This repository is the public VE harness. Sensitive habitat material should stay private and be discussed only through an appropriate private security channel.
